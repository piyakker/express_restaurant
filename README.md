# 餐廳清單
![list image](https://img.onl/okCasR)

## 簡介
可以瀏覽餐廳、查看特定餐廳詳細資訊，以及地圖定位

### 功能
1.瀏覽所有餐廳
2.查看特定餐廳詳細資訊
3.查詢關鍵字來過濾清單
4.連結google map來定位餐廳

## 使用教學
1.確認有安裝 npm 及 node.js
2.將專案 clone 到本地
3.透過終端機 cd 進入資料夾，輸入：
```
npm install
```
4.安裝完畢後，繼續輸入：
```
npm run start
```
5.若看見這行就代表已順利運行，可將網址貼上瀏覽器
```
listening on http://localhost:3000
```
6.暫停使用可於終端機：
```
ctrl + c
```
